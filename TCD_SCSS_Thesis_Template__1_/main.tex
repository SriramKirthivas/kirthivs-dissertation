\chapter{Abstract}
% This dissertation presents a comprehensive survey of cryptographic key reuse and certificate health across Irish mail server infrastructure in 2026. Using ZMap and ZGrab, 19,720 Irish mail servers were discovered and scanned across seven ports to collect TLS certificates and SSH host keys. The SameKeys clustering tool identified 712 key reuse clusters containing 8,756 hosts, a 50\% reduction from the 1,437 clusters found in the 2018 baseline study.

% Beyond replicating the 2018 key reuse analysis, this work extends the investigation with several novel analyses. Certificate health assessment revealed that 26.5\% of certificates are expired and 38\% are self-signed. Let's Encrypt adoption reached 47.1\% overall, with modern ports showing nearly double the adoption rate of legacy SMTP. RC4 cipher analysis found 179 servers still using RC4 eleven years after its deprecation in RFC 7465, with cross-referencing showing that 68\% of these also have expired certificates, indicating abandoned infrastructure. Mail server software fingerprinting identified Postfix as the dominant software at 20.4\%, with managed services like Amazon SES showing significantly better security posture than self-hosted infrastructure. Enhanced cluster visualizations using NetworkX replaced the original Graphviz output, with nodes colored by certificate status and edges simplified by reuse type.

% The findings suggest that Irish mail server security has improved significantly since 2018, driven primarily by Let's Encrypt adoption and the shift toward managed email services, but that substantial pockets of neglected infrastructure remain.
